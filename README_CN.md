# 📝 Notion 每日知识回顾

一个 Agent 技能，将你的 Notion 工作区转化为智能间隔重复系统。每天自动从你的 Notion 中挑选 2-3 个知识点，生成简洁的回顾摘要和深度讨论，并写入专用的 Notion 数据库。

## 功能特性

- **智能选取**：优先推荐与近期 Notion 编辑内容和对话话题相关的知识点，同时加入随机抽取，带来意外发现
- **深度延伸**：不只是简单复习，还会扩展新视角、跨领域关联，并提出发人深省的问题
- **自动初始化**：首次运行自动发现 Notion 结构，识别知识板块，并创建回顾数据库
- **多种风格**：支持 4 种回顾风格——简练有深度、学术探究、轻松对话、苏格拉底式
- **双重输出**：回顾内容同时保存至 Notion（供长期查阅）并在对话中发送（即时阅读）
- **上下文感知**：利用 Agent 记忆，将回顾内容与近期对话和兴趣关联
- **动态发现**：自动识别新增 Notion 板块，无需手动更新配置

## 前提条件

- 支持 MCP 工具调用的 AI Agent
- 包含知识内容的 Notion 工作区（笔记、文章、读书笔记等）
- Notion 集成令牌（[获取方式](#notion-集成配置)）

## 安装

将 `notion-daily-review` 文件夹复制到 Agent 的 skill 目录：

```
<agent-core>/skills/
```

下次对话时，该技能将被自动检测并加载。

## 首次运行

首次使用时，技能会进行交互式初始化：

### 1. Notion 连接检查
技能检查 Notion MCP 工具是否可用并已完成认证。若未配置，将引导你完成 Notion 集成令牌的设置。

### 2. 知识页面发现
技能扫描你的 Notion 工作区，将页面分类为：
- ✅ **知识板块**（笔记、文章、概念等）——纳入回顾
- ❌ **非知识板块**（计划表、追踪器、待办列表）——排除在外

你可以手动调整两个列表中的板块。

### 3. 风格选择
选择你偏好的回顾风格：

| 风格 | 描述 |
|------|------|
| **简练有深度** | 2-3 句核心摘要 + 延伸/新视角，无废话 |
| **学术探究** | 结合框架与理论的结构化分析，附跨学科参考 |
| **轻松对话** | 友好、对话式语气，像和聪明朋友聊有趣的想法 |
| **苏格拉底式** | 以问题为主，极少直接答案，最大化思维激发 |

### 4. 数据库创建
自动在你的 Notion 中创建 `📝 每日知识回顾` 数据库，包含日期、来源板块和触发类型等字段。

### 5. 配置保存
所有设置直接写入技能文件，无需额外配置文件。

## 使用方法

### 手动触发

对你的 Agent 说以下任意一句话：

- `每日回顾` / `知识回顾`
- `daily review` / `notion review`
- `帮我回顾 Notion 知识`

### 定时触发（Cron）

在你的 Agent 中配置定时任务，实现自动触发。例如每天晚上 9 点：

```
schedule: { kind: "cron", expr: "0 21 * * *", tz: "Asia/Shanghai" }
message: "每日回顾"
```

## 工作原理

```
┌─────────────────────────────────────────────┐
│  阶段 1：收集上下文信号                      │
│  • 近期 Notion 编辑（近 3 天）               │
│  • 昨日对话话题（记忆）                      │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│  阶段 2：选取知识点                          │
│  • 1-2 个上下文相关（来自信号）              │
│  • 1 个随机抽取（带来意外发现）              │
│  • 动态页面发现（无硬编码）                  │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│  阶段 3：生成回顾内容                        │
│  • 核心摘要（风格相关）                      │
│  • 深度延伸（扩展 + 问题）                   │
│  • 交叉引用其他 Notion 页面                  │
└──────────────────┬──────────────────────────┘
                   ▼
┌─────────────────────────────────────────────┐
│  阶段 4：输出                                │
│  • 写入 Notion 回顾数据库                    │
│  • 在对话中发送格式化回顾                    │
└─────────────────────────────────────────────┘
```

## Notion 集成配置

1. 访问 [https://www.notion.so/my-integrations](https://www.notion.so/my-integrations)
2. 点击 **"+ 新建集成"**
3. 填写名称（如 "My Agent"），选择工作区，点击 **提交**
4. 复制 **内部集成密钥**（以 `ntn_` 开头）
5. 在每个需要访问的 Notion 页面中：点击 **"..."** → **"连接到"** → 选择你的集成

> **重要**：你只需将集成连接到顶层页面，子页面会自动继承该连接。

## 配置说明

所有配置存储在 `SKILL.md` 文件的 YAML 块中。初始化完成后，内容如下：

```yaml
notion_user_id: "1234567890"
review_database_id: "abc123-def456-..."
review_database_name: "📝 Daily Knowledge Review"
knowledge_root_pages:
  - { id: "page-id-1", name: "知识库" }
  - { id: "page-id-2", name: "项目" }
excluded_pages:
  - "2026 计划表"
  - "习惯追踪器"
agent_memory_path: "/home/user/.agent/memory"
style_preset: "concise_deep"
language: "zh-CN"
```

如需修改设置，可直接编辑 YAML，或对 Agent 说"重新配置每日回顾"。
